 Para más en Mundo del dev

Paginas para aprender :

roadmap

Hojas de ruta para desarrolladores roadmap.sh es un esfuerzo comunitario para crear hojas de ruta, guías y otros contenidos educativos que ayuden a guiar a los desarrolladores en elegir un camino y guiar sus aprendizajes.

w3schools

w3schools es un sitio web con recursos para aprender las diferentes tecnologías del front-end, si recién estás comenzando o ya tienes algo de experiencia sin duda w3schools te seguirá ayudando para mejorar.

developer.mozilla

developer.mozilla es otro sitio web con guías en Html, Css y Javascript y te enseñará paso a paso cómo ser un experto en estas tecnologías, además te da un plan de estudio para los nuevos desarrolladores front-end, visítalo.

manz.dev

manz.dev es un blog que ofrece contenido educativo en tecnologías web, con tutoriales detallados y consejos prácticos para desarrolladores de todos los niveles. Visítalo para mejorar tus habilidades y aprender las novedades del desarrollo web.

freecodecamp.org

freecodecamp.org es una plataforma educativa gratuita que ofrece cursos interactivos para aprender a programar. Con una comunidad global de apoyo, puedes mejorar tus habilidades en HTML, CSS, JavaScript y otras tecnologías. Obtén certificaciones y colabora en proyectos reales para ganar experiencia práctica. Ideal para principiantes y desarrolladores de todos los niveles.

css-tricks

css-tricks.com es un recurso valioso para desarrolladores web, ofreciendo artículos, videos y guías sobre CSS y otras tecnologías web. Aprende técnicas avanzadas, sigue tutoriales paso a paso y mantente al día con las últimas tendencias en el desarrollo web.

microfrontend.dev

microfrontend.dev es una plataforma educativa que ofrece recursos y guías sobre la arquitectura de microfrontends. Aprende cómo dividir una aplicación web en partes más pequeñas y manejables, facilitando el desarrollo y la mantenimiento. Ideal para desarrolladores que buscan mejorar sus habilidades y adoptar nuevas tecnologías.

web.dev

web.dev es una plataforma educativa creada por Google que ofrece recursos, guías y herramientas para mejorar la calidad de los sitios web. Aprende sobre rendimiento, accesibilidad, seguridad y SEO para crear experiencias web excepcionales. Ideal para desarrolladores de todos los niveles que buscan perfeccionar sus habilidades.

Diseño inspiración. UX/UI

figma community

www.figma.com/community es una plataforma donde los diseñadores pueden compartir y explorar recursos de diseño. Encontrarás plantillas, íconos, y otros recursos útiles creados por la comunidad. Únete para colaborar y mejorar tus habilidades de diseño con Figma.

Iconos, formas para tu web :

boxicons

boxicons.com es un sitio que ofrece una colección de íconos vectoriales personalizables para tus proyectos web. Con una amplia variedad de estilos y categorías, boxicons.com te permite mejorar el diseño y la usabilidad de tus sitios web y aplicaciones de manera sencilla.

wweb.dev

wweb.dev/resources/css-separator-generator es una herramienta en línea que te permite crear separadores CSS personalizados para tus proyectos web. Con una interfaz fácil de usar, puedes generar y personalizar separadores de manera rápida y eficiente. Perfecto para diseñadores y desarrolladores que buscan mejorar el estilo visual de sus sitios web.
